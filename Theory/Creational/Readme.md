CREATIONAL DESIGN patterns
- Effect system of product creation
- ability to create object in controlled and organized way

 Types
 - Singleton pattern (example - single database connection)
 - Factory method pattern (example - create car from carfactory)
 - Abstract Factory pattern (example - create table , sofas, chair in funiture store)
 - Builder pattern (step by step addition)
 - Prototype pattern (cloning of existing product)
